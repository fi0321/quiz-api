Title: Login

Primary Actor: User: A Student/Instructor

Stakeholders and Interests:

User: The users in these cases are the students taking the quiz and the instrtuctors administrating it

Precondition: The users are registered in the system and given different level of access depending on the type of user

  1. User logs into the system.
  2. If the credentials are verified the user is given access to the dashboard
  3. The type of dashboard will depend on the type of user, a student or instructor
