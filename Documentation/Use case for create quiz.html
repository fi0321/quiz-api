Title: Create Quiz

Primary Actor: User:Instructor

Stakeholders and Interests:

User: The users in this case are the instrtuctors creating it

Precondition: The users are logged in the system as an instructor

Main Scenario:

  1. User logs into the system.
  2. If the credentials are verified the user is given access to the instructor dashboard
  3. Once inside the dashboard the instructor can add questions from the question bank to the quiz
  4. The instructor can set start and end time for the quiz
  5. Once the quiz is created, it is ready to be attempted.
  6. The questions can be removed from the quiz and new questions can be added.
  7. The entire quiz can be deleted.

Extensions:

1a. If the user is not registered as an instructor, it is not given access to create a quiz.
